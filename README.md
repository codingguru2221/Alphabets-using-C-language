# ASCII Alphabet Printer

This project is a simple C++ program that prints the letters A to Z in an ASCII art style. Each letter is represented by a 7x7 grid of asterisks (`*`) and spaces.

## Getting Started

### Prerequisites

To compile and run this program, you need a C++ compiler. You can use any standard C++ compiler like `g++`.

### Compilation

To compile the program, navigate to the directory containing the `main.cpp` file and run the following command:

```bash
g++ -o ascii_printer main.cpp
```

This will compile the code and create an executable file named `ascii_printer`.

### Running the Program

To run the compiled program, use the following command:

```bash
./ascii_printer
```

This will execute the program and print the ASCII art for each letter from A to Z to the console.
